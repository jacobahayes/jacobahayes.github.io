About Me

Projects

These are group projects I have worked on with classmates

Garage Sail

Summer 2016

Garage Sail is a THE ultimate yard sale helper. It serves as a dashboard for managing yard/garage/estate sales. Built using the Play framework for Java with a mySQL database. Garage Sail is capable of posting to social media to get the word out about your upcoming sales, searching for similar items on Ebay, and has a host of other features built in as well. Source code can be found on Github.

GT Movie

Summer 2016

GTMovie is a web app for purchasing tickets to movies ahead of time, much like the popular website Fandango. It was built with the Flask web framework for Python and has a mySQL database in the backend. Feel free to check out the source code on Github.

Joint

Spring 2017 (Ongoing)

Joint is a platform to perform multi-origin flight searches to an optimal destination. It finds the destination for the group travelling based on origin cities and parameters such as price, speed, activity, attractions, etc. This will be useful for friends, families, and colleagues who live in different locations who want to meet at a common location. Currently, we are still in the design phase of this project. More on this to come.
