Check out my work

Garage Sail

Garage Sail is a THE ultimate yard sale helper. It serves as a dashboard for managing yard/garage/estate sales. Built using the Play framework for Java with a mySQL database. Garage Sail is capable of posting to social media to get the word out about your upcoming sales, searching for similar items on Ebay, and has a host of other features built in as well. Source code can be found on GitHub.

GTMovie

GTMovie is a web app for purchasing tickets to movies ahead of time, much like the popular website Fandango. It was built with the Flask web framework for Python and has a mySQL database in the backend. Feel free to play around with the application and check out the source code on GitHub.
